# Network Monitor v1.0


## About

A python tkinter based gui application to test your network. 

You can test:

 1. Ping
 2. Download Speed
 3. Upload Speed


## Built Using

 - Python
 - Tkinter
 - Speedtest CLI
 - PySpeedTest

## Modules Required

Python Modules Required for this project includes:

 - Tkinter
 - pyspeedtest
 - speedtest

```sh
pip install tkinter
```

```sh
pip install pyspeedtest
```
```sh
pip install speedtest-cli
```
